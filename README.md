# 🏟️ Sports Booking Landing Page

A bold, high-impact sports court booking landing page built with pure HTML and CSS — developed as **Task 1** of the Oasis Infobyte Web Development Internship.


---

## 📸 Project Overview

This project is a visually striking **single-page landing site** for a fictional sports court booking platform. It covers Football Turf, Badminton, and Cricket booking — designed with a dark athletic aesthetic, neon-green accents, and strong typographic hierarchy.

The goal was to demonstrate core frontend skills: layout structuring, section design, responsive spacing, CSS visual effects, and creative use of color and typography — all without any JavaScript or frameworks.

---

## 🎯 Features

- **Hero Section** — Full-viewport background with blur + dark overlay effect, large display typography, and sport category cards with hover animations
- **Facilities Section** — Two-column CSS Grid layout with statistics and feature highlights using colored left-border accents
- **Footer / CTA Section** — Full-height call-to-action with a clean divider and city availability list
- **Hover Interactions** — Button ghost effect on hover, card lift with neon glow on sport cards
- **Google Fonts Integration** — Oswald + Roboto Condensed for a bold athletic typographic system

## 🧠 Key Concepts Practiced

- CSS `::before` / `::after` pseudo-elements for layered background effects
- CSS Grid for two-column responsive section layout
- `position: relative` / `z-index` stacking for hero overlay
- CSS `transition` for smooth hover interactions
- `background-image`, `background-size: cover`, `filter: blur()` for visual depth
- Consistent design system using a neon-green (`#c4ff00`) accent color
